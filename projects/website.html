This Website

It may seem obvious, but this website is one of my projects—and one that I'm really proud of! I built it as a way to showcase my other work and to give myself a space to experiment with design and development. Right now, it's pretty simple, built with just HTML, CSS and some light JavaScript, but I wanted to keep things clean and functional while making sure it reflects my style. Along the way, I experimented with some features—like a dark mode toggle which added a nice touch to the overall experience.

Of course, a project like this is never truly "finished." I'm always looking for ways to improve it—whether that's adding new features, tweaking the design, or even exploring more advanced web technologies. As I continue working on new projects, this site will grow and evolve along with them. Stay tuned for updates!
