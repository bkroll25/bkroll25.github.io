This Website

It may seem obvious, but this website is one of my projects—and one that I'm proud of! I haven't worked with HTML or CSS since Highschool, so it was a fun challenge to get back into it. I built it as a way to showcase my other work and to give myself a space to experiment with design and development. Right now, it's pretty simple, built with just HTML, CSS and a little JavaScript, but I wanted to keep things clean and functional while making sure it reflects my style. I did experiment with some features—like my dark mode toggle, but other than that it is just the basics

Of course, a project like this is never truly "finished." I'm always looking for ways to improve it—whether that's adding new projects, or tweaking the design. As I continue working on new projects, this site will grow and evolve along with them. Stay tuned for updates!

Check out the source code on my GitHub!
